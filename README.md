# Flow Assistant (Chrome Extension)

Flow Assistant เป็น Chrome Extension ที่สร้างขึ้นมาเพื่อช่วยให้คุณสามารถสั่งงานและสร้าง Prompt สำหรับ AI บนหน้าเว็บ `labs.google/fx/tools/flow` ได้อย่างสะดวก รวดเร็ว และเป็นระบบมากยิ่งขึ้น โดยรองรับการอ่านและแนบรูปภาพอ้างอิงหลายหมวดหมู่ พร้อมฟีเจอร์ปรับแต่งพร้อมท์ให้อัตโนมัติ

## 🌟 ฟีเจอร์เด่น (Features)

- **🖼️ ระบบจัดการรูปภาพอ้างอิง (Multi-Slot Storage)**: แยกหมวดหมู่สินค้า (Product), นางแบบ (Face), เครื่องแต่งกาย (Clothes) และสถานที่ (Location) ทำให้ใช้งานได้ง่ายและแม่นยำขึ้น
- **🛍️ โหมด Commercial Ads**: สร้างโฆษณาสินค้าแบบมืออาชีพ โดยมีตัวเลือกสไตล์ที่หลากหลาย เช่น Skincare, Fashion, Food, 3D Cartoon ฯลฯ พร้อมปิด/เปิดการใช้นางแบบประกอบ
- **🪄 AI สร้างคำโฆษณา (AI Gen Text Graphic)**: ใช้ Gemini AI ในการคิดคำโปรยโฆษณาที่โดนใจสำหรับวางบนรูปภาพ
- **📸 ตั้งค่ามุมกล้องแบบเจาะลึก**: เลือกกล้อง, เลนส์, รูรับแสง (Aperture), Shot Size และมุมกล้องได้ตามต้องการ
- **🔄 ระบบ Batch Generation**: ตั้งรันการสร้างภาพอัตโนมัติต่อเนื่องได้หลายรอบ โดยไม่ต้องกดรันเองทีละครั้ง

## 🛠️ การติดตั้ง (Installation)

เนื่องจากยังไม่ได้พับลิชบน Chrome Web Store คุณสามารถติดตั้งเพื่อใช้งานได้ด้วยวิธี Developer Mode ดังนี้:

1. ดาวน์โหลดโปรเจกต์นี้และแตกไฟล์ Zip ออกมา หรือดึงโค้ดลงเครื่องผ่าน Git
2. เปิด Google Chrome แล้วพิมพ์ URL `chrome://extensions/`
3. เปิดสวิตช์ **Developer mode** ที่มุมขวาบน
4. คลิกปุ่ม **Load unpacked** (โหลดส่วนขยายที่เปิดเครื่องหมายไว้)
5. เลือกโฟลเดอร์ `flow-assistant` ที่คุณดาวน์โหลดมา
6. เสร็จสิ้น! กดที่ไอคอน Extension บน Toolbar เพื่อเปิดหน้าต่าง Sidepanel ขึ้นมาใช้งานได้เลย

## 🚀 วิธีใช้งานเบื้องต้น (Usage)

1. **เปิดหน้าเว็บเป้าหมาย**: ไปที่ [labs.google/fx/tools/flow](https://labs.google/fx/tools/flow)
2. **เปิด Flow Assistant**: กดที่ไอคอน Extension (หรือตั้งค่าให้อยู่ในโหมด Side Panel ของ Chrome เสมอ)
3. **เลือกโหมดการทำงาน**: เลือกระหว่าง Standard หรือ Commercial Ads ตามงานที่คุณต้องการ
4. **เพิ่มรูปอ้างอิง (ถ้ามี)**: อัปโหลดรูปภาพที่ต้องการให้ AI อ้างอิงในแต่ละหมวดหมู่
5. **ปรับแต่งรายละเอียด**: ใส่คำสั่งเพิ่มเติม, เลือกสไตล์, ปรับรายละเอียดนางแบบ, และตั้งค่ามุมกล้องให้ครบถ้วน
6. **ส่งคำสั่ง**:
   - กด **ดูล่วงหน้า (Preview)** เพื่อเช็คโครงสร้าง Prompt
   - หรือกด **ส่งคำสั่ง (Generate)** เพื่อรันสคริปต์กรอก Prompt และกดปุ่มสร้างให้บนหน้าเว็บแบบอัตโนมัติ 

## ⚙️ โครงสร้างไฟล์ (File Structure)

- `manifest.json`: ไฟล์คอนฟิกหลักของ Chrome Extension สำหรับตั้งชื่อ รุ่น และขอสิทธิ์ต่างๆ (Permissions: `sidePanel`, `activeTab`, `scripting`, `storage`)
- `sidepanel.html`: เลย์เอาต์และ UI (HTML & CSS) สำหรับแสดงผลใน Side Panel ด้านข้าง
- `sidepanel.js`: สคริปต์ฝั่ง UI ควบคุมการอัปโหลดรูป, เก็บ Data ลง Local Storage และควบคุมจังหวะการทำ Batch
- `prompts.js`: รวมความรู้ (Knowledge) โครงสร้างการเรียบเรียง Prompt และคลังตัวเลือก เช่น Pose, สไตล์งาน ฯลฯ
- `background.js`: Worker ขนาดเล็กที่ช่วยจับ Event เมื่อกดปุ่ม Extension ให้เด้งเปิดเป็น Side Panel ทันที

## 📝 ข้อควรระวัง (Notes)

- สคริปต์จะพยายามจำลองการพิมพ์ข้อความผ่าน Slate.js ลงไปในกล่องข้อความบนเว็บ หากหน้าตาเว็บไซต์มีการเปลี่ยนแปลง อาจต้องเข้ามาอัปเดตตัว Selector
- ไฟล์รูปภาพที่ผู้ใช้อัปโหลดจะถูก Resize ก่อนบันทึกลง Local Storage เพื่อประหยัดพื้นที่ โดยเก็บไว้เฉพาะบนบราวเซอร์ของคุณเท่านั้น
- AI Text Gen จำเป็นต้องขอ Gemini API Key หนึ่งครั้งสำหรับการเรียกใช้งาน AI ในการคิดคำ
